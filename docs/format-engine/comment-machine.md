## Why Comments Matter

### 1. Algorithm Signal
Platforms (YouTube, Instagram, TikTok) prioritize content with high engagement. Comments = engagement = more reach.

### 2. Accountability Mechanism
Public commitment increases follow-through. Commenting DONE creates social accountability.

### 3. Data Collection
Comments become your dataset. "Energy before: 3/10, after: 7/10" = measurable impact. Aggregate these, and you have proof the system works.

### 4. Community Identity
Repeat commenters become recognizable. They're not just viewers‚Äîthey're participants in a shared lab.

## CTA Keyword System

Every format uses a **CTA keyword**‚Äîa short, memorable word viewers comment to signal completion.

### Examples

| Keyword | Format | Action |
|---------|--------|--------|
| SHAKE | Micro-Mission | Shake hands for 10 seconds |
| TIMER | Micro-Mission | 5-minute focus timer |
| PHONEFREE | Real Youth Problem | Phone outside bedroom, 3 nights |
| EARLYBIRD | Weekly YouTube | Wake at 5 AM for 7 days |
| JAB | Skills Track | Practice jab in air, 10 reps |

### Why Keywords Work

**Trackability:** You can search comments for "DONE" or "SHAKE" and count participation.

**Simplicity:** One word is easier to remember than "Comment if you completed this."

**Community Language:** Keywords become inside language. "I'm on day 5 of EARLYBIRD" = shared experience.

**Gamification:** Leaderboards based on keyword count ("Who commented the most keywords this month?").

## Producer Notes

### Filming the CTA
- **On-screen text:** Show the keyword in bold (e.g., "Comment SHAKE")
- **Voiceover:** Say it clearly: "Comment SHAKE plus your energy rating, one to ten."
- **Repetition:** Mention the CTA twice (mid-video reminder + end-screen)

### Editing the CTA
- **End screen card:** Text overlay with keyword and rating scale
- **Example comment:** Show an example on screen: "DONE. Energy 7/10."
- **Visual cue:** Use color or animation to highlight the CTA section

### Tracking Comments

**Manual Method:**
- Weekly spreadsheet: Count keywords per video
- Note repeat usernames (identify core community)
- Track rating distributions (most common energy level, focus level, etc.)

**Automated Method (Future):**
- YouTube Analytics API (export comments)
- Python script to parse and count keywords
- Dashboard showing keyword trends over time

### Mistake 2: No Pinned Comment
Pinning the CTA makes it the first thing viewers see. Don't skip this.

### Mistake 3: Ignoring Comments
If you don't reply, viewers stop commenting. Engagement is a two-way street.

### Mistake 4: Changing Keyword System
Don't invent new formats every week. Stick to DONE + keyword + rating. Consistency = trackability.

### Mistake 5: Not Celebrating Participation
Shout out the people who show up. Leaderboards, video mentions, pinned comments‚Äîrecognition fuels repetition.
